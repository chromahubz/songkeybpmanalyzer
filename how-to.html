How To Use the Song Key & Tempo Analyzer

A quick tutorial to analyze tracks and create a smooth, harmonic mix order.

Quick Start

  1. Open the Analyze section on the home page.
  2. Drag and drop your audio files into the drop zone, or click to select files.
  3. Watch as each track is analyzed for key (Harmonic Wheel mapping) and BPM.
  4. When you have 2+ tracks, click Create Mix to generate the optimal order.
  5. Review the sequence and transitions (Perfect Match, Energy Boost, etc.).

Manual Entry (No File Upload)

Have a list of songs and want a quick harmonic order without uploading audio? Use the Manual Entry tab on the home page to add tracks with title, artist, key (Harmonic Wheel code like 8B), and BPM. The app will include these in the mix order just like analyzed files.

Import a JSON Song List

If you exported your list earlier, you can import it on the home page's Import Songs tab. The tool loads the songs and generates an optimal sequence from that list.

Tips for Harmonic Mixing

  • Same-number A/B swaps (e.g., 8A ↔ 8B) are smooth key changes.
  • Adjacent numbers with same letter (8A → 9A) tend to blend well.
  • Small BPM differences (±5) often need minimal tempo adjustment.
  • Energy Boost transitions can increase set intensity tastefully.
